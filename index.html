A. Overview

Getting started — login, dashboard orientation, and org configuration

Login to Zygal Admin Portal Getting Started
Access the portal via Chrome or Edge browser at https://platform.zygal.io
  1. Open your web browser (Chrome or Edge recommended) on a laptop or desktop
  2. Type https://platform.zygal.io in the address bar and press Enter
  3. Locate the Email / Username field and enter your registered admin email ID
  4. Click the Password field and enter your secure password (check that CAPS LOCK is off)
  5. Complete the CAPTCHA verification if shown on screen
  6. Click the Login button — you will be redirected to the Home dashboard automatically
Portal Login Screen — platform.zygal.io
Ensure you have valid admin credentials before attempting to log in. Contact your system administrator if login fails.
Home — Overview Dashboard Dashboard
After login, the Home page displays a real-time summary of your entire surveillance infrastructure.
  • Review summary cards/tiles showing total devices, active devices, and inactive devices
  • Identify red or warning indicators highlighting offline devices or configuration problems
  • Use the dashboard for quick situational awareness before diving into specific modules
Home Dashboard — Overview of device status, alerts, and system health
Organisation Details Configuration
Configure your organisation's profile, location, roles, and email server from the Organisation Details section.
  1. Click Organisation Details from the menu to open your org configuration
  2. In Basic Details: verify/update Organisation Name, Website URL, Contact Number, and Logo
  3. Scroll to Location Details: set the correct Time Zone, Country, State, City, and Address
  4. In the Roles section: review and add/edit roles (e.g. Admin, Supervisor, Viewer)
  5. Configure the Email Server: choose Zygal's server or enter your own SMTP details (Host, Port, Username, Password)
Organisation Details — Basic configuration and location settings
Security Settings Security
Enable Two-Factor Authentication and configure network firewall restrictions for enhanced security.
  • Under Admin Basic Details: verify First Name, Last Name, Contact Number, and License Key
  • In the Security section: enable Two Factor Authentication (2FA) for an extra security layer on admin logins
  • Optionally configure Network Firewall restrictions to allow portal access only from specific IPs
2FA is strongly recommended for production environments to prevent unauthorized access.

Add New Device Device Setup
Register new hardware and virtual devices to the Zygal platform — step-by-step process.
  1. Go to Device & Management → Add Device and click on Zygal
  2. Select the device type you want to add (e.g. Link Device) and click Add New
  3. In the next window, click Add Device again
  4. Obtain a License Key from the device license list under Utility — enter it in the form
  5. Fill in all required device details and click Save
  6. Select the newly added device and click Plot to position it on the map
  7. Enter Latitude and Longitude (get these from Google Maps) in the search bar
  8. Click Plot, place the pointer on the exact location, then click Save

Add Device — License
A valid license key is required before adding any device. Obtain this from your Zygal account manager.
